CRYPTOLAB - Визуализатор математических шифров
===============================================

Версия: 1.0
Дата выпуска: 2026

О ПРОГРАММЕ:
------------
CryptoLab - это образовательное приложение для изучения основ
криптографии и математического шифрования. Программа наглядно
демонстрирует работу различных алгоритмов шифрования.

ВОЗМОЖНОСТИ:
------------
✓ Шифр Цезаря с интерактивным слайдером
✓ Шифр Виженера с ключевым словом
✓ RSA с демонстрацией генерации ключей
✓ Протокол Диффи-Хеллмана
✓ XOR-хеширование
✓ Пошаговое отображение математических вычислений

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
---------------------
• Windows 7/8/10/11
• 100 МБ свободного места
• 2 ГБ ОЗУ

УСТАНОВКА:
----------
1. Запустите установочный файл CryptoLab_Setup.exe
2. Следуйте инструкциям установщика
3. Запустите программу с рабочего стола или из меню Пуск

ИСПОЛЬЗОВАНИЕ:
--------------
1. Выберите алгоритм шифрования в левой панели
2. Введите текст для шифрования
3. Настройте математические параметры в правой панели
4. Нажмите "Зашифровать" или "Расшифровать"
5. Наблюдайте пошаговые вычисления в нижней панели
